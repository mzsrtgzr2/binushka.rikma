## 5) Header + Footer Redesign Mockups

### Header Spec
- Height:
  - desktop 88px
  - mobile 72px
- Logo left, nav center/right, CTA right.
- Sticky with subtle surface and border on scroll.
- Mobile drawer:
  - grouped links
  - contact shortcut
  - social links at bottom

### Footer Spec
- Desktop 4-column information architecture:
  1. Brand + contact
  2. Workshops/services
  3. Store/resources
  4. Legal/newsletter
- Mobile: collapsible groups + persistent WhatsApp action.

## 6) Visual Spec Snapshot
- Palette: warm neutral base + terracotta/gold brand accent.
- Typography: Assistant/Heebo pairing with generous line-height.
- Corners: 12-18 radius for cards and controls.
- Sections: predictable vertical rhythm and clear title/description pattern.
